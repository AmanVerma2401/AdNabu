# Shopify Store AdNabu Automation using Selenium

This project automates an e-commerce Shopify store AdNabu using Selenium WebDriver with Python.

The automation script performs the following actions:
- Opens Shopify AdNabu store
- Enters store password
- Searches for a product
- Selects the product
- Adds product to cart

---

# 🚀 Technologies Used

- Python
- Selenium WebDriver
- Chrome Browser
- ChromeDriver

---

# 📋 Prerequisites

Install the following before running the project.

## 1. Install Python

Download Python:

https://www.python.org/downloads/

---

## 2. Install PyCharm Community Edition

Download PyCharm:

https://www.jetbrains.com/pycharm/download/

---

## 3. Install Google Chrome

Download Chrome Browser:

https://www.google.com/chrome/

---

## 4. Install Selenium

Open terminal or command prompt and run:

pip install selenium

---

# ▶️ How to Run the Project

1. Open project in PyCharm
2. Copy the automation script
3. Run the Python file
4. Automation will:
   - Open Shopify website
   - Enter password
   - Search product
   - Add product to cart
